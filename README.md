# Customer Churn Prediction

## Overview
This project predicts customer churn using Machine Learning techniques. The goal is to identify customers who are likely to leave a service based on their demographics, account information, and usage patterns.

## Dataset Files
- data.csv – Original dataset
- cleaned_data.csv – Preprocessed dataset
- churn_cleaned_final.csv – Final dataset used for modeling

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Feature Selection
5. Model Building
6. Model Evaluation

## Models Used
- Logistic Regression
- Decision Tree Classifier

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report
- ROC-AUC Score
